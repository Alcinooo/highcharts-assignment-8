Test and check: First chart with Highcharts (with CDN)

This is an example of a simple chart embedded in a Web page. The chart data came from Highcharts: Your First Chart. The Highcharts Basic Bar Chart demo is here — it uses the latest version of the JavaScript code. Note that the chart's bars become thicker if you add height to the CSS for your container DIV.

Charts are usually placed somewhere in the middle of an article for which the chart provides additional information. Most numerical data is easier for the reader to understand when it appears in chart form. Choosing the best chart type to suit your data is very important when you want to communicate the information effectively.
